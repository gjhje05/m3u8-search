# 🔍 m3u8 搜

> 输入关键词，搜索内网视频站点，自动解析 m3u8 播放地址。支持多线程下载 & 批量管理，相当于一个轻量级 m3u8 下载器。

## ✨ 功能特色

### 🔎 搜索
- **一键搜索** — 输入关键词，爬取目标站点，解析每个视频的 m3u8 地址
- **智能分页** — 自动识别总页数，上一页/下一页自由翻页
- **勾选复制** — 每个 m3u8 地址前有复选框，勾选后一键复制（Excel 格式）
- **目录自定义** — 可输入目录名，自动追加到每行末尾，翻页保留不丢失
- **图片预览** — 点击封面图放大查看，加载失败可点击重试
- **Excel 友好** — 复制格式为 `地址\t名称\t目录`，直接粘贴到 Excel 自动分列

### 📥 下载管理（独立页面 `/downloads`）

支持 **多任务并行下载**，m3u8 切片智能合并，媲美专业下载器：

- **分段并行下载** — 默认 36 线程并行下载所有 .ts 切片，requests.Session 复用 TCP 连接
- **多任务并发** — 可配置同时运行的任务数（默认 3），超出自动排队等待
- **进度可视化** — 实时进度条、下载速度、文件大小展示
- **分段网格视图** — 每个分片的下载状态用彩色方块直观展示（⬜ 待下载 ✅ 成功 ❌ 失败）
- **重试机制** — 失败的分段自动重试，支持单独重试失败分段
- **强制合并** — 即使部分分段下载失败，也能合并已有分段
- **智能回退** — 分段过少时自动切换到 ffmpeg 直接下载
- **等待队列** — 超过并发限制的任务进入等待队列，完成后自动启动
- **任务管理** — 四标签页（下载中/等待中/已完成/失败），可删除等待任务、清空队列
- **可配置参数** — 保存目录、下载线程数、并发任务数均可调整

### 🧵 下载引擎特点
- 多线程（可配置 1-128 线程）分段下载
- 60 秒 m3u8 解析缓存，避免 CDN 限流
- ffmpeg concat 无损合并为 MP4
- 分段下载失败时自动重试（应对 CDN sign 过期）
- 持久化任务列表（重启不丢失）

## 🛠️ 技术栈

| 层 | 技术 |
|------|------|
| **后端** | Python 3.14 + Flask + requests + BeautifulSoup + lxml |
| **前端** | HTML + CSS + JS（文件分离，全部带中文注释） |
| **爬取** | 伪装浏览器请求 → BS4 解析 HTML → 正则提取 .m3u8 链接 |
| **下载** | 多线程分片下载 + ffmpeg concat 合并 |
| **运行** | 本地运行，局域网可访问 |

## 🚀 快速开始

### 前提条件
- Python 3.8+
- [FFmpeg](https://ffmpeg.org/download.html)（下载合并需要，可在 PATH 或 WinGet 安装）

### 安装运行

```bash
# 1. 安装依赖
pip install flask requests beautifulsoup4 lxml

# 2. 运行
python app.py

# 3. 打开浏览器
# http://127.0.0.1:5000          # 搜索页
# http://127.0.0.1:5000/downloads  # 下载管理页
```

### 修改目标站点

编辑 `app.py` 中的 `TARGET_BASE_URL`：

```python
TARGET_BASE_URL = "http://192.168.50.188"   # 改成你的站点地址
```

## 📋 搜索页复制格式

勾选后复制到剪贴板的内容格式：

```
https://xxx.com/video.m3u8    视频名称前10字    目录名
https://yyy.com/play.m3u8     另一个视频名     目录名
```

每列用 **Tab** 分隔，粘贴到 Excel / WPS 表格自动分成三列。

## ⚙️ 功能详解

### 搜索
输入关键词后按 Enter 或点击搜索按钮，后端自动请求目标站点的搜索页面并解析。

### 翻页
搜索结果超过一页时，顶部显示「上一页 / 下一页」按钮。

### 勾选 & 复制
- ☐ 勾选想要的行
- ☐ 全选：顶部「☐ 全选」一键勾选全部
- 点击「📋 复制选中」复制所有勾选项

### 图片
- 点击封面图 → 放大预览（点击遮罩层或 ESC 关闭）
- 图片加载失败 → 显示「加载失败，点击重试」

### 下载
在搜索页勾选后点击「⬇️ 下载选中」，输入目录名（可选），跳转到下载管理页。

下载管理页（`/downloads`）功能：
- **四标签页**：下载中 / 等待中 / 已完成 / 失败
- **等待队列**：超过并发限制的任务自动排队，完成后自动启动
- **实时进度**：进度条 + 速度 + 文件大小实时更新
- **分段网格**：每个 .ts 分片状态一目了然
- **操作按钮**：取消下载、打开文件夹、重试失败分段、强制合并

### 并发配置
- **线程数**（1-128）：每个任务的分片并行下载数
- **并发任务**（1-20）：同时运行的最大任务数
